Chrome学习笔记
2020-04-23

一、九大功能面板

  • Elements元素面板
    • 检查和调整页面，调试DOM，调试CSS
  • Network网络面板
    • 抓包调试网络请求，了解页面静态资源分布，网页性能检测
  • Console控制台面板
    • 调试JS，查看Console log日志，交互式代码调试
  • Sources源代码资源面板
    • 调试JS页面源代码，断点调试
  • Application应用面板
    • 查看&调试客户端存储，如Cookie，LocalStorage，SessionStorage等
  • Performance性能面板
    • 查看页面性能细节，细粒度对网页载入进行性能优化（高阶）
  • Memory内存面板
    • JS CPU分析器，内存堆分析器（高阶）
  • Security安全面板
    • 查看页面安全及证书问题
  • Audits面板
    • 使用Google Lighthouse辅助性能分析，给出优化建议（高阶）

二、打开快捷键

  1. 打开最近关闭的状态：Cmd+Opt+I
  2. 快速查看DOM或样式：Cmd+Opt+C
  3. 快速进入Console查看log运行JS：Cmd+Opt+J
  4. F12打开，同1

三、Console

访问DOM节点

document.querySelectorAll(‘节点标签’)

使用$0可引用focus的元素

右键Copy -> Copy JS Path，贴到Console中即可引用对应元素

DOM断点调试

  • 属性修改时
  • 节点删除时
  • 子树修改时

右键Break on -> subtree子树修改 / attribute属性修改 / node节点删除

属性修改：console中利用JS Path选择这个节点而后.属性=“新值” 回车；页面被锁定；点击继续即进行断点修改

子树修改&节点删除都可进行节点删除：console中利用JS Path var 待删变量名 = 这个节点；利用JS Path选择父元素.removeChild(待删变量名); 回车执行
